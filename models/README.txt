Place your converted 3D model here as 'pod.glb'

Current file: Custom PODS MVP.SLDPRT
Required format: GLTF (.gltf) or GLB (.glb)

Conversion steps:
1. Export SLDPRT from SolidWorks as STL or VRML
2. Import into Blender or use online converter
3. Export as GLB format
4. Name the file 'pod.glb' and place it in this folder

The web app will automatically load pod.glb when you open index.html
